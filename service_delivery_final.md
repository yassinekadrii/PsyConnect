## 1. Platform Scale & Scope
- **Web Application:** 9 Core Functional Pages (Admin, Doctor, Patient, Auth modules).
- **Mobile Application:** 22 Functional Screens (Across 4 dedicated user modules).
- **Backend API:** Over 30 Secure REST Endpoints & Real-time Signaling Hub.

---

## 2. Technical Architecture Stack
- **Mobile Frontend:** React Native with Expo (Native Modules).
- **Web Frontend:** HTML5, CSS3 (Glassmorphism), Vanilla JavaScript.
- **Backend:** Node.js with Express Framework.
- **Database:** MongoDB (NoSQL) for high-flexibility data.
- **Real-time Engine:** Socket.io for chat and WebRTC for Video/Audio.
- **Security:** JWT Session Tokens & Bcrypt Password Hashing.

---

## 3. Detailed Feature Breakdown

### A. Patient Features (22 Mobile Screens)
- **Integrated Mood Tracker:** Track daily emotional states with visual logs.
- **Real-time Messaging:** Instant chat with assigned doctors.
- **Video & Audio Calling:** Peer-to-peer encrypted consultations.
- **Medical Records:** View and download digital prescriptions (PDF).
- **Profile Management:** Update bio, phone, and profile pictures.

### B. Doctor Features (Web & Mobile)
- **Patient Management:** View patient list, history, and mood logs.
- **Prescription Creator:** Digital tool to generate prescriptions instantly.
- **Professional Profile:** Manage specialties, bio, and availability.
- **Real-time Consultations:** High-quality video/audio interface.

### C. Admin & Security (9 Web Pages)
- **Platform Dashboard:** Monitor user registrations and platform stats.
- **User Control:** Validate and manage Doctor/Patient accounts.
- **Security:** 6-digit Email Verification required for all accounts via SMTP.
